# How to Create Popup Box Visuals

## How to Edit BI for Intune Reports
### Creating a Popup Effect in Power BI

Many PowerStacks customers have asked how to make visuals in Power BI feel more interactive—specifically, how to create a “popup” effect when users click an element. This short tutorial walks through the exact steps to achieve that effect using shapes, buttons, and layer ordering in Power BI. Follow along to see how you can add a dynamic, polished experience to your reports without writing any code.
