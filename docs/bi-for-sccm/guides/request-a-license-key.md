# Request a Trial License Key

Request a fully functional 30-day trial license key for BI for SCCM. You should receive the key within 10 minutes of submitting the form. If you do not see the email, please check your junk folder.

Only one key per email domain will be generated. If you or someone else from your organization has previously requested a key, contact us at [biforsccm@powerstacks.com](mailto:biforsccm@powerstacks.com) for assistance.
