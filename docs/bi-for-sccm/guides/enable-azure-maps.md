# Enable Azure Maps in Power BI
Several of the pages in the out-of-the-box reports contain the map visual. If the map visuals do not display in your environment they might be disabled. To ensure that the maps visual is enabled follow this guide.

### Step 1





1. In **Power BI** select the **Settings** icon.
1. Select **Admin portal**.
![](images/Admin_Portal_settings-1-465x1024.png)
### Step 2





1. Select **Tenant settings.**
1. In the **Integration settings**section expand **Use Azure Maps visual.**
1. Select **Enabled.**
1. Select **Apply.**
![](images/Enable_Maps-1.png)
